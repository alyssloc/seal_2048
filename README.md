# Seal and Sea Lion 2048

A 2048-style puzzle game built in Python/Pygame where numbers are replaced by 15 NOAA-tracked seal and sea lion species ([NOAA Site](https://www.fisheries.noaa.gov/seals-sea-lions#overview)). Used pygbag to convert into WebAssembly for instant play on GitHub Pages.
